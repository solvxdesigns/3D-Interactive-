# HAI - Humanoid Artificial Intelligence (Solvx Designs)

This repository contains the source code for the HAI humanoid chat interface, developed under Solvx Designs. It features cinematic 3D robotic animations and a modern web frontend.

## Overview

HAI provides an interactive experience combining fluid 3D experiences with AI interaction.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/solvxdesigns/3D-Interactive-.git
   cd 3D-Interactive-
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

### Application Details

- **Frontend**: React, Vite
- **3D**: Three.js, React Three Fiber
- **Styling**: TailwindCSS
