# HAI Chatbot Website - Complete Replit Prompt

Create a cinematic single-page AI chatbot website with scroll-triggered 3D robot animations that transform into a functional chat interface.

## Design System

**Colors:**
- Background: #050505 (dark void), #000000 (pure black for hero)
- Accents: #00F2FF (cyan), #FF00E5 (magenta)
- Text: #FFFFFF (primary), rgba(255,255,255,0.7) (muted)
- Glass: rgba(20,20,20,0.6) with blur(40px)

**Typography:** 
- Font: 'Orbitron' or 'Exo 2' (futuristic sans-serif)
- Sizes: H1 120px, H2 48px, H3 32px, Body 18px, Small 14px
- Letter-spacing: 0.05em-0.3em

**Effects:**
- Glassmorphism: backdrop-filter blur(40px), rgba backgrounds, 1.5px white borders (20% opacity)
- Gradients: linear-gradient(135deg, #00F2FF, #FF00E5)
- Glows: box-shadow 0 0 20px rgba(0,242,255,0.6)

### Section 4: Robot Fall & Chat Interface (STICKY)

**Trigger:** When scroll reaches 300vh (start of section)

#### PHASE 1: Robot Fall Animation

**Step 1 - Loss of Balance (1s):**
```
Robot wobbles left/right
Rotation Z: -15° → +15° → -10° → +10° (decreasing amplitude)
Eyes change from (• •) to (X X)
Arms flail randomly
```

**Step 2 - The Fall (1.5s):**
```
Position Y: current → +1000px (falls below screen)
Rotation: 0° → 720° (2 full spins)
Scale: 1.0 → 0.8 (perspective)
Easing: Gravity-like curve
```

**Step 3 - Impact (0.2s):**
```
Screen shake: ±3px random, 4 iterations
White flash overlay: 0 → 0.3 → 0 opacity
Optional sound: Thud effect
```

#### PHASE 2: Shatter & Transform

**Step 4 - Shatter (1s):**
```
Robot explodes into 20 geometric fragments:
- Shapes: Triangular/polygonal 3D pieces
- Size: Random 30-80px
- Colors: White with cyan/magenta gradient edges
- Motion: Radial burst outward
- Physics: Each shard rotates (100-300°/s) and falls

CRITICAL: Robot HEAD stays INTACT
- Does NOT shatter
- Tumbles separately
- Glows brighter cyan
```

**Step 5 - Magnetic Assembly (1.5s):**
```
After 0.5s pause:
- All 20 fragments reverse direction
- Fly toward screen center (bezier curves)
- Speed increases as approaching
- Rotation slows to 0°
- Form horizontal chat bar shape (800×70px)
- Fragments overlap and merge with glow

Robot Head transformation:
- Shrinks: scale 1.0 → 0.3 (over 800ms)
- Moves to RIGHT side of chat bar
- Morphs into microphone icon
- Eyes remain visible (now tiny)
- Settles with gentle bounce
```

### Scroll Implementation Logic
```javascript
// Pseudo-code structure

const scrollY = window.scrollY;
const vh = window.innerHeight;

// Section boundaries
const section1 = 0 to vh;        // Hero
const section2 = vh to vh*2;     // Robot intro
const section3 = vh*2 to vh*3;   // Features
const section4 = vh*3+;          // Chat (sticky)

// Robot animation controller
if (scrollY >= vh && scrollY < vh*2) {
  const progress = (scrollY - vh) / vh; // 0-1
  
  if (progress < 0.25) {
    // Robot enters
    robotY = lerp(-500, 0, progress/0.25);
    robotRotY = lerp(0, 360, progress/0.25);
  }
  else if (progress < 0.5) {
    // Floating idle
  }
  else if (progress < 0.75) {
    // Gestures
  }
  else {
    // Move to side
    robotX = lerp(0, -30vw, (progress-0.75)/0.25);
  }
}

// Trigger fall animation once
if (scrollY >= vh*3 && !fallTriggered) {
  triggerRobotFall();
  fallTriggered = true;
}
```
